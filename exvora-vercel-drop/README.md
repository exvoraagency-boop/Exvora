# EXVORA Agency — Website

Marketing agency site built with Next.js 14 (App Router), ready for Vercel.

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Deploy to Vercel
1. Push this folder to a GitHub repo.
2. In Vercel: New Project → import the repo (Next.js auto-detected).
3. Deploy. Then add your domain `exvoraagency.com` under Project → Settings → Domains.

## Contact form
The form opens the visitor's email client to `hello@exvoraagency.com`.
To receive submissions server-side instead, connect a service like Formspree,
Resend, or a Next.js route handler.

## Brand assets
All in `public/images/` (logo + banners you provided).
